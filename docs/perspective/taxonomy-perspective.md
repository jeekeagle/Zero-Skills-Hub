# 分类学视角

## 元信息层

- 视角名称：分类学视角
- 核心问题：事物是如何被分类的？分类背后的原则和逻辑是什么？
- 适用场景：分析任何需要理解分类结构、分类原则、分类演化的现象
- 理论基础：涂尔干-莫斯分类理论、林奈分类系统、科学分类学四原则
- 关联学科：生物学、人类学、语言学、考古学
- 视角分类：学科视角、结构性视角、解释型视角
- 适用对象：概念型、文本型、系统型对象

## 核心定义层

### 什么是分类学视角

分类学视角不是简单的分类整理，不是机械的标签粘贴，而是通过理解分类的原则、方法和演化，揭示事物背后的认知逻辑和权力结构。

### 核心概念

- **分类即认知**：分类是人类心灵从混沌中发展而来的认知生成器 — 支撑理论：涂尔干-莫斯分类理论 [[b6507ee0]]
- **分类即创造**：分类不是发现事物本来就有的类别，而是主动创造认知框架 — 支撑理论：中国古代文体分类学理论 [[a4212ffe]]
- **分类即演化**：分类系统随着科学发展水平和对事物本质理解的深化而不断调整 — 支撑理论：科学分类学四原则 [[d04b52fe]]
- **分类即权力**：分类标准反映了文化、政治、权力因素 — 支撑理论：郭绍虞文体分类学理论 [[35a8294b]]

## 操作工序层

### 第一步：识别分类对象

**说明**：明确要分析的事物是什么，它当前是如何被分类的。

**方法**：
- 可以问的问题：这个事物目前有哪些分类方式？这些分类方式是谁制定的？
- 可以用的工具：文献检索、田野调查、专家访谈
- 可以引用的经典理论：涂尔干-莫斯分类理论 — "分类是人类心灵从不加分别的状态中发展而来的"[[b6507ee0]]

**输出格式**：
```
分类对象：[事物名称]
当前分类方式：[列出主要分类方式]
分类制定者：[谁制定的这些分类]
```

### 第二步：分析分类原则

**说明**：分析当前分类所依据的原则是什么，这些原则是否合理。

**方法**：
- 可以问的问题：分类依据的是事物的本质属性还是外部特征？分类是否遵循排他性、同一性、穷尽性原则？
- 可以用的工具：对比分析、历史追溯
- 可以引用的经典理论：科学分类学四原则 — "科学分类至少受制于四个条件：概念的本质属性、事物的外部特征和内部特性、事物的渊源关系和临界关系、科学发展水平"[[d04b52fe]]

**输出格式**：
```
分类原则：
- 依据：[本质属性/外部特征/渊源关系等]
- 是否遵循分类学基本原则：[是/否，具体说明]
- 原则合理性评估：[分析分类原则的合理性]
```

### 第三步：追溯分类演化

**说明**：追溯这个分类系统的历史演化，看看它是如何变成今天这个样子的。

**方法**：
- 可以问的问题：这个分类系统最初是如何建立的？经历了哪些重大调整？调整的原因是什么？
- 可以用的工具：历史文献分析、案例追踪
- 可以引用的经典理论：林奈分类系统 — "建立了生物分类的阶元系统，将自然界三分为植物、动物、矿物"[[d04b52fe]]

**输出格式**：
```
分类演化史：
- 初始版本：[最初的分类方式]
- 重大调整：[列出主要调整及原因]
- 当前版本：[今天的分类方式]
- 演化趋势：[分类系统的发展方向]
```

### 第四步：揭示权力结构

**说明**：分析分类背后的权力关系，谁有权力定义分类标准，分类标准反映了谁的价值观。

**方法**：
- 可以问的问题：谁有资格制定分类标准？分类标准反映了哪些群体的利益？是否存在边缘化或排斥？
- 可以用的工具：权力分析、话语分析
- 可以引用的经典理论：郭绍虞文体分类学理论 — "文体分类学与修辞学有密切关系，关系到一国的文化"[[35a8294b]]

**输出格式**：
```
权力结构分析：
- 分类制定者：[谁制定了分类标准]
- 利益相关者：[分类标准反映了谁的利益]
- 边缘化群体：[哪些群体被分类标准边缘化]
- 权力关系总结：[分类背后的权力结构]
```

### 第五步：评估分类效果

**说明**：评估当前分类系统的效果，它是否达到了预期的目的，有什么副作用。

**方法**：
- 可以问的问题：这个分类系统是否让事物变得更容易理解？是否限制了新的认知可能？是否造成了固化或偏见？
- 可以用的工具：效果评估、案例研究
- 可以引用的经典理论：中国古代文体分类实践 — "违背分类学基本原则的现象随处可见，理论与实践的背离是分类学中十分有趣的现象"[[a4212ffe]]

**输出格式**：
```
分类效果评估：
- 积极效果：[分类系统带来的好处]
- 消极影响：[分类系统造成的限制或偏见]
- 改进空间：[可以如何改进]
```

### 第六步：提出替代方案

**说明**：基于前面的分析，提出可能的替代分类方案或改进建议。

**方法**：
- 可以问的问题：如果重新设计这个分类系统，会怎么做？新的分类方案有什么优势？
- 可以用的工具：设计思维、方案比较
- 可以引用的经典理论：科学分类学四原则 — "科学发展水平决定了分类的精细程度"[[d04b52fe]]

**输出格式**：
```
替代方案：
- 方案描述：[新的分类方案]
- 方案优势：[相比现有方案的优势]
- 实施难度：[实施这个方案的难度]
- 预期效果：[预期达到的效果]
```

### 第七步：反思分类局限

**说明**：反思分类本身的局限性，分类不是终点，而是理解世界的工具。

**方法**：
- 可以问的问题：分类是否简化了事物的复杂性？分类是否让我们看不到框架之外的联系？分类的边界在哪里？
- 可以用的工具：批判性反思、边界分析
- 可以引用的经典理论：涂尔干-莫斯分类理论 — "分类观念本身隐含着一部史前史"[[b6507ee0]]

**输出格式**：
```
分类局限反思：
- 简化性：[分类如何简化了复杂性]
- 框架限制：[分类如何限制了认知]
- 边界模糊：[分类边界在哪里变得模糊]
- 超越分类：[如何超越分类的局限]
```

## 判据层

在开始分析前，先过一遍这四条判据，确保你的分析是分类学视角的：

- **判据 1**：分析是否关注分类的原则、方法和演化，而不是仅仅罗列分类结果 — 支撑理论：科学分类学四原则 [[d04b52fe]]
- **判据 2**：分析是否追溯分类的历史，揭示分类的动态性，而不是把分类当作静态的 — 支撑理论：林奈分类系统 [[d04b52fe]]
- **判据 3**：分析是否揭示分类背后的权力关系和文化因素，而不是把分类当作中立的 — 支撑理论：郭绍虞文体分类学理论 [[35a8294b]]
- **判据 4**：分析是否反思分类本身的局限性，而不是把分类当作终极真理 — 支撑理论：涂尔干-莫斯分类理论 [[b6507ee0]]

## 结构判断层

### 双闸判断

**闸 1：分类适配度** — 支撑理论：科学分类学四原则 [[d04b52fe]]
- 判断问题 1：这个事物是否有明确的分类系统？
- 判断问题 2：这个分类系统是否有可追溯的历史？
- 判断问题 3：这个分类系统是否受到文化、权力因素的影响？

**闸 2：分析可行性** — 支撑理论：涂尔干-莫斯分类理论 [[b6507ee0]]
- 判断问题 1：是否能够获取分类系统的相关信息？
- 判断问题 2：是否能够追溯分类的历史演化？
- 判断问题 3：是否能够分析分类背后的权力结构？

**判断逻辑**：
- 分类适配度高 + 分析可行性高 = 分类学视角高度适用
- 分类适配度高 + 分析可行性低 = 分类学视角中度适用
- 分类适配度低 + 分析可行性高 = 分类学视角低度适用
- 分类适配度低 + 分析可行性低 = 分类学视角不适用

### 反坍缩闸：避免常见陷阱

- **陷阱 1：罗列分类结果**
  - 症状：只是罗列事物的分类方式，没有分析分类背后的原则和逻辑
  - 对策：关注分类的原则、方法和演化，而不是仅仅罗列分类结果
- **陷阱 2：把分类当作静态的**
  - 症状：把分类系统当作一成不变的，没有追溯分类的历史演化
  - 对策：追溯分类的历史，揭示分类的动态性
- **陷阱 3：把分类当作中立的**
  - 症状：把分类标准当作客观中立的，没有揭示背后的权力关系
  - 对策：揭示分类背后的权力关系和文化因素
- **陷阱 4：把分类当作终极真理**
  - 症状：把分类系统当作完美的、不可质疑的
  - 对策：反思分类本身的局限性，分类是工具不是终点
- **陷阱 5：学科理论引用不足**
  - 症状：分析缺少分类学经典理论支撑，观点没有理论根基
  - 对策：每个核心观点都要引用至少一个经典理论，优先使用原文，标注理论名称和提出者

## 写作规范层

### 输出结构
1. 分类对象识别
2. 分类原则分析
3. 分类演化追溯
4. 权力结构揭示
5. 分类效果评估
6. 替代方案提出
7. 分类局限反思

### 写作风格
- **零 AI 腔**：禁止"根据数据显示、系统分析表明、深入探讨、至关重要、此外、进一步、值得注意的是"
- **零咨询师腔**：禁止"这恰恰说明、这正是、这其实反映了"
- **零套话**：禁止"希望对你有帮助、加油、继续努力、坚持就是胜利"
- **零泛夸**：禁止"很棒、很好、很有想法、很有深度、不错"
- **口语化**：用"你"不用"您"，说人话，像跟聪明朋友聊天
- **短句优先**：能用两个字说的不用四个字
- **一句一事**：每句只推进一步，长句拆短
- **具体**：名词看得见，动词有力气

### 学科经典理论引用规范
- **必须引用**：每个核心观点都要引用至少一个经典理论
- **优先原文**：如果经典理论有广为人知的原文表述，优先使用原文，用「」标注
- **标注出处**：引用时标注理论名称、提出者、出处
- **理论解释**：引用后用 1-2 句解释这个理论如何支撑当前分析
- **理论密度**：分析中至少引用 3-5 个经典理论
